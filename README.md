# 🛒商品管理システム

## 機能一覧
* 商品の購入（在庫を減らす）
* 商品追加
* 商品の更新
* 商品削除

## 環境
バックエンド：SpringBoot

フロントエンド：React、BootStrap

DB：H2Console

ユニットテスト：JUnit

E2Eテスト：PlayWright

## 実行方法
SpringBoot（http://localhost:8080）
``` bash
mvn spring-boot:run
```

React（http://localhost:5173）
``` bash
npm run dev
```

H2Console（http://localhost:8080/h2-console/）

ユーザー名：`sa` / パスワード：`password`
