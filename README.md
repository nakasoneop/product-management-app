# 🛒商品管理アプリ

## 機能一覧
* 商品の購入（在庫を減らす）
* 商品追加
* 更新
* 削除

## 環境
バックエンド：SpringBoot

フロントエンド：React

DB：H2Console（http://localhost:8080/h2-console/）

ユニットテスト：JUnit

E2Eテスト：PlayWright

## サーバー起動
SpringBoot（http://localhost:8080）
``` bash
mvn spring-boot:run
```

React（http://localhost:5173）
``` bash
npm run dev
```
